# 算法篇

- 基础数据结构(抛开语言)

  1. 数据
     数据连续，支持随机访问（访问第 n 个数据），根数据在内存里的存储方式比较像
     新增数据的复杂度 o(n)
     查找数据 根据索引查 o(1)
     值查找，只能遍历查找 o(n)
  2. 链表
     链表就是非连续存储
     新增删除，o1 复杂度
     查找，On
     链表，react-fiber。vue-keep-alive 中使用的 lru 算法就是使用链表

     环形链表判断会否有环

  3. 对象（哈希表） 数组+链表

     增加、删除、查询复杂度都是 O(1)

  4. 树

     dom 节点就是一个树，链表分叉
     二叉树，二叉树的最大深度，树形结构使用地推

     ```js
     // 二叉树递归
     // 多叉树可以简化为二叉树
     var maxDepth = function (root) {
       if (root === null) {
         return 0;
       }
       return Math.max(root);
     };
     ```

     两棵树是否相同
